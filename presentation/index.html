MongoDB is a NoSQL database which stores the data in form of key-value pairs. It is an Open Source, Document Database which provides high performance and scalability along with data modelling and data management of huge sets of data in an enterprise application.

Overview of MongoDB

MongoDB consists of a set of databases. Each database again consists of Collections. Data in MongoDB is stored in collections.

We can use namespace to logically group and nest collections

There can be one collection named

db.studytonight.users

to save user informations, then there can be others like

db.studytonight.forum.questions

and

db.studytonight.forum.answers

to store forum questions and answers respectively

Data is stored and queried in BSON, its binary representation of JSON-like data.
